Welcome to Quotinator v1.0 readme.txt

-------------------------------------

You can click anywhere on the console program to type commands.

Accepted commands are:

Quote -- Displays a random quote to the console window

Tweet -- Opens an external page to tweet the last quote received (you do not have to be logged on to access this functionality)

Help -- Displays the help menu (similar to this readme)

Info -- Contains info about the quotinator program

-------------------------------------

About the Quotinator:

The quotinator was developed by Dylan Parker in response to a FreeCodeCamp.com project. It's sole purpose is to provide the user with random quotes and give them the ability to share those quotes via twitter.
